# AI-Powered Smart Interview Simulation & Feedback Engine

A full-stack web application built with Django that simulates real interview 
scenarios, evaluates candidate responses using AI, and generates structured feedback.

## Tech Stack
- Python, Django
- HTML, CSS, JavaScript
- AI API Integration
- SQLite

## Features
- Dynamic interview question generation
- AI-powered answer evaluation
- Structured feedback reports
- User session management

## How to Run
pip install -r requirements.txt
python manage.py runserver
